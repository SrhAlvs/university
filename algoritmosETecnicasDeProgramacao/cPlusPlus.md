# Bibliotecas
## `#include <iostream>`
* **i** = input (entrada)
* **o** = output (saída)
* **stream** = fluxo, no caso, vamos trabalhar com fluxos de entradas e saídas

# Comandos
* `std` = *standart*
* `cout` = *out command* (comando de saída)
* `endl` = *end line*
* `int` = integer (inteiro), cria uma variável inteira
